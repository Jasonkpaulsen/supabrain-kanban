# SupaBrain Kanban

Mobile-first Progressive Web App kanban board backed by Supabase.

## Features
- Bottom tab navigation between status columns
- Swipe cards right/left to advance/move back status
- Long-press to edit, tap to expand description
- Pull-to-refresh
- Offline support with background sync
- iOS safe area support (notch + home indicator)
- Desktop mode with drag-and-drop at 769px+

## Install on iOS
1. Open the GitHub Pages URL in Safari
2. Tap the Share button
3. Select "Add to Home Screen"
